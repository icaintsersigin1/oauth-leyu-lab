# oauth-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面，作为静态资源的存储与展示用途。

## 项目简介

这是一个以 HTML 页面为主要内容的归档仓库，包含若干独立页面。仓库本身不针对特定域名或网站，仅作为页面资源的集合与版本管理。

## 目录说明

- 仓库根目录下直接存放各 HTML 文件。
- 每个 HTML 文件均为一个独立页面，互不依赖。
- 可根据需要按目录组织页面，具体结构以实际提交为准。

## 页面归档说明

- 所有页面均为静态 HTML，不包含后端逻辑。
- 页面资源（如 CSS、JavaScript、图片等）建议内联或与 HTML 文件一同提交。
- 页面内容可能随时间更新或调整，历史版本可通过 Git 记录追溯。

## 维护说明

- 欢迎提交 Issue 反馈页面问题或提出建议。
- 如需新增页面，请直接提交 Pull Request，并附上简要说明。
- 仓库维护者将定期检查并合并合理的贡献。

## 许可证

本仓库内容遵循项目根目录下的 LICENSE 文件。若无明确说明，则默认保留所有权利。
